Find...

Opens up a search dialog which will move the cursor to search finds in the current file.

Find in Files...

Opens up a search dialog which will search through all open files and present the results in the "Find Results" tab at the bottom of the screen. These results are clickable and will show the find position.

Search Again

This option will repeat the last search command.

Replace...

Opens up a dialog that can replace text by text in the current file.

Incremental Search

Opens up a text box that performs an search-as-you-type search, just like in browsers like Chrome and Firefox.

Goto Function...

Brings up a dialog that lists all available functions and allows you to search through them and navigate to them by clicking.

Goto Line...

Brings up a dialog that allows one to enter a line number and navigate to it.
